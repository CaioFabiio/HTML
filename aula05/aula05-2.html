Tabelas

Uma tabela é definida com a tag <table>.

Cada linha da tabela é definida com a tag <tr>.

Uma célula de cabeçalho é definida por <th>. Por padrão, linhas de cabeçalho são centralizadas e em negrito.

Uma célula é definida com a tag <td>.

Exemplo

              <!DOCTYPE html>
              <html>
                <head>
                  <title>Exemplo de Tabela</title>
                  <meta charset="utf-8">
                </head>
              
                <body>
                  <table style="width:100%">
                    <tr>
                      <th>Nome</th>
                      <th>Sobrenome</th>
                      <th>Idade</th>
                    </tr>
                    <tr>
                      <td>Júlia</td>
                      <td>Lisboa</td>
                      <td>25</td>
                    </tr>
                    <tr>
                      <td>Ricardo</td>
                      <td>Oliveira</td>
                      <td>28</td>
                    </tr>
                  </table>
                </body>
              </html>

Para visualizar as bordas, utiliza-se o atributo border.

Exemplo

              <table border="1"></table>

Mesclar Células

O atributo colspan é utilizado para mesclar a célula em mais de uma coluna.

Exemplo

              <!DOCTYPE html>
              <html>
                <head>
                  <title>Exemplo de Tabela</title>
                  <meta charset="utf-8">
                </head>
              
                <body>
                  <table border="1">
                    <tr>
                      <th>Nome</th>
                      <th colspan="2">Telefone</th>
                    </tr>
                    <tr>
                      <td>Bill Gates</td>
                      <td>55577854</td>
                      <td>55577855</td>
                    </tr>
                  </table>
                </body>
              </html>

O atributo rowspan é utilizado para mesclar a célula em mais de uma linha.

Exemplo

              <!DOCTYPE html>
              <html>
                <head>
                  <title>Exemplo de Tabela</title>
                  <meta charset="utf-8">
                </head>
              
                <body>
                  <table border="1">
                    <tr>
                      <th>Nome</th>
                      <td>Bill Gates</td>
                    </tr>
                    <tr>
                      <th rowspan="2">Telefone</th>
                      <td>55577854</td>
                    </tr>
                    <tr>
                      <td>55577855</td>
                    </tr>
                  </table>
                </body>
              </html>

Título da Tabela

A tag <caption> define um título para a tabela. Deve ser adicionada imediatamente após a tag <table>

Exemplo

              <!DOCTYPE html>
              <html>
                <head>
                  <title>Exemplo de Tabela</title>
                  <meta charset="utf-8">
                </head>
              
                <body>
                  <table border="1">
                    <caption>Economia Mensal</caption>
                    <tr>
                      <th>Mês</th>
                      <th>Economia</th>
                    </tr>
                    <tr>
                      <td>Janeiro</td>
                      <td>R$100,00</td>
                    </tr>
                    <tr>
                      <td>Fevereiro</td>
                      <td>R$50,00</td>
                    </tr>
                  </table>
                </body>
              </html>

Resumo

  • <table> define uma tabela
  • <tr> define uma linha
  • <td> define uma célula
  • <th> define uma célula de cabeçalho
  • <caption> define um título para a tabela
  • border define as bordas da tabela
  • colspan mescla uma célula em várias colunas
  • rowspan mescla uma célula em várias linhas
