Links

Links são encontrados em todas as páginas web. Links permitem que os usuários naveguem de uma página a outra.

A tag <a> define um link.

              <a href="url">Texto do link</a>

Exemplo

              <a href="http://www.google.com/">Google</a>

O atributo href especifica o endereço de destino do link (http://www.google.com).

O Texto do Link é a parte visível do link na página.

Links Locais

O exemplo acima utilizou um endereço da internet.

Um link local é especificado com o endereço do arquivo.

Exemplo

              <a href="exemplo.html">Exercício Anterior</a>

Atributo Target

O atributo target especifica onde abrir o link.

target pode ter um dos seguintes valores:

  • _blank abre o link em uma nova aba ou janela
  • _self abre o link na mesma aba (padrão)

Exemplo

              <a href="http:www.google.com/" target="_blank">Google</a>

Imagens como Links

Para utilizar uma imagem como link, basta inserir a tag <img> dentro da tag <a>.

Exemplo

              <a href="link">
                <img src="imagem.jpg" alt="Descrição da imagem" style="width:480px;height:360px;">
              </a>

Título do Link

O atributo title especifica uma informação extra sobre o elemento.

É visualizado quando o mouse fica sobre o link.

Exemplo

              <a href="http:www.google.com/" title="Buscar no Google">Google</a>

Marcador

Marcadores são usados para permitir que o usuário possa pular para uma parte específica da página. Podem ser bastante úteis em páginas longas.

Para usar um marcador, primeiro é necessário criá-lo, e então adicioná-lo ao link.

Exemplo

Primeiro, cria-se o marcador com o atributo id:

              
              <h3 id="C4">Capítulo 4</h3>

Adiciona-se então um link ao marcador, na mesma página:

              
              <a href="#C4">Ir para o Capítulo 4</a>

Ou adiciona-se um link ao marcador, a partir de outra página:

              
              <a href="arquivo.html#C4">Ir para o Capítulo 4</a>
Exemplo

Resumo

  • <a> define um link
  • href especifica o endereço do link
  • target define onde o link será aberto
  • <img> dentro de um <a> para usar uma imagem como link
  • id="valor" define marcadores na página
  • href="#valor" link para o marcador
