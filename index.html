O Que é HTML?

HTML é o formato padrão para criação de páginas web.

  • HTML é abreviação para Hyper Text Markup Language
  • HTML Descreve a estrutura de páginas Web utilizando marcação
  • Elementos HTML são a base das páginas HTML
  • Elementos são representados por tags
  • Tags rotulam partes do conteúdo, como "título", "parágrafo", "tabela", etc
  • Os navegadores não exibem as tags do HTML, mas as utilizam para formatar o conteúdo da página

Um Exemplo HTML

              <!DOCTYPE html>
              <html>
                <head>
                  <title>Titulo da Pagina</title>
                </head>
              

                <body>
                  <h1>Meu primeiro titulo</h1>
                  <p>Meu primeiro paragrafo</p>
                </body>
              </html>
  • A declaração <!DOCTYPE html> define que o arquivo será HTML5
  • O elemento <html> é a base de uma página HTML
  • O elemento <head> contém metainformações sobre o arquivo
  • O elemento <title> especifica um título para o arquivo
  • O elemento <body> contém o conteúdo visual da página
  • O elemento <h1> define um cabeçalho grande
  • O elemento <p> define um parágrafo

Tags

Tags em HTML são elementos entre < >:

<tag>conteúdo...</tag>

  • Tags geralmente são em pares como <p> e </p>
  • A primeira tag do par é a tag inicial ou tag de abertura, a segunda tag é a tag final ou tag de fechamento
  • A tag de fechamento é escrita como a tag inicial, porém adicionando uma barra antes do nome da tag

Navegadores

O propósito de um Navegador de Internet (Chrome, Firefox, Internet Explorer) é ler documentos HTML e exibi-los.

O navegador não exibe as tags HTML, mas as usa para determinar como exibir o documento.

Estrura de Uma Página HTML

Abaixo está a visualização da estrutura de uma página HTML:

<html>
<head>
<title>Título da Página</title>
</head>
<body>
<h1>Isto é um cabeçalho</h1>
<p>Isto é um parágrafo.</p>
<p>Isto é outro parágrafo.</p>
</body>
</html>

Nota: Apenas o conteúdo dentro da seção <body> (a área branca acima) é exibido no navegador.

Declaração <!DOCTYPE>

A declaração <!DOCTYPE> representa o tipo do documento, e ajuda o navegador a exibir as páginas corretamente.

Deve aparecer apenas uma única vez, antes de qualquer tag.

              <!DOCTYPE html>

Documentos HTML

Todos os documentos HTML devem começar com a declaração do tipo do documento: <!DOCTYPE html>.

O documento HTML em si começa com <html> e termina com </html>.

A parte visível do documento HTML fica entre <body> e </body>.
