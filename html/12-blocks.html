Elementos de Bloco e Linha

« Listas CSS »

Todo elemento tem um valor padrão de exibição dependendo do tipo de elemento. O padrão para a maioria dos elementos é inline ou bloco.

Elementos de Bloco

Um elemento de bloco sempre se inicia em uma nova linha e preenche todo o espaço disponível na linha em que está.

O elemento <div> é um elemento de bloco.

O elemento <div> geralmente é utilizado como um container para outros elementos. Pode ser usado para estilizar blocos de conteúdo.

Exemplo

              <!DOCTYPE html>
              <html>
                <head>
                  <title>Elemento &lt;div&gt;</title>
                  <meta charset="utf-8">
                </head>
              
              
                <body>
                  <div style="background-color:black;color:white;padding:20px;">
                    <h2>Tóquio</h2>
                    <p>Tóquio é a capital do Japão. Situa-se em Honshu, a maior ilha do arquipélago.</p>
                  </div>

                </body>
              </html>

Elemento de Linha

Um elemento de linha não se inicia em uma nova linha e somente preenche o espaço necessário.

Este é o elemento de linha <span> dentro de um parágrafo.

O elemento <span> é utilizado como container para textos. Pode ser utilizado para estilizar textos.

Exemplo

              <!DOCTYPE html>
              <html>
                <head>
                  <title>Elemento &ltspan&gt</title>
                  <meta charset="utf-8">
                </head>
              
              
                <body>
                  <div style="background-color:black;color:white;padding:20px;">
                    <h2>Tóquio</h2>
                    <p>Tóquio é a capital do <span style="color:red;">Japão</span>. Situa-se em Honshu, a maior
              ilha do arquipélago.</p>
                  </div>

                </body>
              </html>
