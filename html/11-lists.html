Listas Desordenadas

Uma lista desordenada começa com a tag <ul>. Cada item da lista é um elemento <li>.

Os itens serão marcados com pontos pretos por padrão:

Exemplo

              <ul>
                <li>Café</li>
                <li>Leite</li>
                <li>Chá</li>
              </ul>

Alterando os Marcadores

A propriedade list-style-type é utilizada para definir o estilo do marcador de itens da lista:

Valor Descrição
disc Define o marcador como um ponto (padrão)
circle Define o marcador como um círculo
square Define o marcador como um quadrado
none Os marcadores não são exibidos

Ponto

              <ul style="list-style-type:disc;">
                <li>Café</li>
                <li>Leite</li>
                <li>Chá</li>
              </ul>

Círculo

              <ul style="list-style-type:circle;">
                <li>Café</li>
                <li>Leite</li>
                <li>Chá</li>
              </ul>

Quadrado

              <ul style="list-style-type:square;">
                <li>Café</li>
                <li>Leite</li>
                <li>Chá</li>
              </ul>

Sem Marcador

              <ul style="list-style-type:none;">
                <li>Café</li>
                <li>Leite</li>
                <li>Chá</li>
              </ul>

Listas Ordenadas

Uma lista ordenada começa com a tag <ol>. Cada item da lista é um elemento <li>.

Os itens da lista serão marcados com números por padrão:

Exemplo

              <ol>
                <li>Café</li>
                <li>Leite</li>
                <li>Chá</li>
              </ol>

Atributo type

O atributo type da tag <ol> define o tipo de marcador de item da lista:

Tipo Descrição
type="1" Os itens serão ordenados com números (padrão)
type="A" Os itens serão ordenados com letras maiúsculas
type="a" Os itens serão ordenados com letras minúsculas
type="I" Os itens serão ordenados com numerais romanos maiúsculos
type="i" Os itens serão ordenados com numerais romanos minúsculos

Números

              <ol type="1">
                <li>Café</li>
                <li>Leite</li>
                <li>Chá</li>
              </ol>

Letras Maiúsculas

              <ol type="A">
                <li>Café</li>
                <li>Leite</li>
                <li>Chá</li>
              </ol>

Letras Minúsculas

              <ol type="a">
                <li>Café</li>
                <li>Leite</li>
                <li>Chá</li>
              </ol>

Numerais Romanos Maiúsculos

              <ol type="I">
                <li>Café</li>
                <li>Leite</li>
                <li>Chá</li>
              </ol>

Numerias Romanos Minúsculos

              <ol type="i">
                <li>Café</li>
                <li>Leite</li>
                <li>Chá</li>
              </ol>

Contador da Lista

Por padrão, uma lista ordenada começará a contar a partir do 1. O atributo start inicia a contagem a partir de um número especificado:

Exemplo

              <ol start="50">
                <li>Café</li>
                <li>Leite</li>
                <li>Chá</li>
              </ol>

Listas Aninhadas

Listas podem ser aninhadas (listas dentro de listas):

Exemplo

              <ul>
                <li>Café</li>
                <li>Chá
                  <ul>
                    <li>Chá Preto</li>
                    <li>Chá Verde</li>
                  </ul>
                </li>
                <li>Leite</li>
              </ul>

Nota: Itens de listas podem conter novas listas, e outros elementos, como imagens e links, etc.
