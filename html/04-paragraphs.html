Parágrafos

O elemento <p> define um parágrafo:

Exemplo

              <p>Isto é um parágrafo.</p>
              <p>Isto é outro parágrafo.</p>

Quebra de Linha

O elemento <br> define uma quebra de linha.

<br> é utilizado para iniciar uma nova linha sem inciar um novo parágrafo:

Exemplo

              <p>Isto é<br>um parágrafo<br>com quebra de linha.</p>

A tag <br> é uma tag vazia, ou seja, não tem uma tag de fechamento.

O Problema do Poema

Este poema será exibido em uma única linha:

Exemplo

              <!DOCTYPE html>
              <html>
                <head>
                  <title>Texto Pré-Formatado</title>
                  <meta charset="utf-8">
                </head>
              
              
                <body>
                  <h1>Texto sem pré-formatação:</h1>
                  <p>
                    Minha terra tem palmeiras,
                    Onde canta o sabiá;
                    As aves que aqui gorjeiam,
                    Não gorjeiam como lá.
                  </p>

                </body>
              </html>

Texto Pré-Formatado

O elemento <pre> define um texto pré-formatado.

O texto no elemento <pre> é exibido em fonte monoespaçada (geralmente Courier), e mantém tanto os espaços como quebra de linhas.

Exemplo

              <!DOCTYPE html>
              <html>
                <head>
                  <title>Texto Pré-Formatado</title>
                  <meta charset="utf-8">
                </head>
              
              
                <body>
                  <h1>Texto pré-formatado:</h1>
                  <pre>
                    Minha terra tem palmeiras,
                    Onde canta o sabiá;
                    As aves que aqui gorjeiam,
                    Não gorjeiam como lá.
                  </pre>

                </body>
              </html>
