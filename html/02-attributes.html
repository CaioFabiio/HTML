Atributos fornecem informações adicionais sobre os elementos.

Atributos

  • Todos os elementos HTML podem ter atributos
  • Atributos fornecem informações adicionais sobre um elemento
  • Atributos sempre são especificados na tag de abertura
  • Atributos geralmente vêm em pares nome/valor: nome="valor"

Atributo href

Links são definidos com a tag <a>. O endereço do link é especificado no atributo href:

Exemplo

              <a href="https://www.google.com/">Isto é um link</a>

Atributo title

Aqui, o atributo title adicionado ao elemento <p>. O valor do atributo title será exibido quando o mouse estiver sobre o parágrafo:

Exemplo

              <p title="Sou um título">Isto é um parágrafo.</p>

Atributo alt

O atributo alt especifica um texto alternativo a ser usado quando uma imagem não puder ser exibida.

Exemplo

              <img src="imagem.jpg" alt="Texto da imagem.">

Atributo style

O atributo style é usado para especificar a estilização de um elemento, como a cor, fonte, tamanho etc.

Exemplo

              <p style="color:red;">Isto é um parágrafo</p>
