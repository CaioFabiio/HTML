Cabeçalhos

Cabeçalhos são definidos com as tags de <h1> a <h6>.

<h1> define o cabeçalho mais importante. <h6> define o cabeçalho menos importante.

Exemplo

              <h1>Cabeçalho 1</h1>
              <h2>Cabeçalho 2</h2>
              <h3>Cabeçalho 3</h3>
              <h4>Cabeçalho 4</h4>
              <h5>Cabeçalho 5</h5>
              <h6>Cabeçalho 6</h6>

Cabeçalhos São Importantes

Motores de busca utilizam cabeçalhos para indexar a estrutura e o conteúdo das páginas web.

Usuários selecionam as páginas pelo título. É importante usar cabeçalhos para exibir a estrutura do documento.

Cabeçalhos <h1> devem ser usados para títulos principais, seguidos por cabeçalhos <h2> e assim por diante.

Nota: Utilize cabeçalhos apenas para títulos. Não utilize cabeçalhos para fazer textos GRANDES ou negrito.

Régua Horizontal

A tag <hr> indica uma mudança temática em uma página. É exibida como uma linha horizontal.

O elemento <hr> é usado para separar o conteúdo em uma página HTML:

Exemplo

              <h1>Este é o cabeçalho 1</h1>
              <p>Algum texto.</p>
              <hr>
              <h2>Este é o cabeçalho 2</h2>
              <p>Outro texto.</p>
              <hr>
