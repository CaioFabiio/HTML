Elementos HTML

Um elemento HTML normalmente consiste de uma tag de início e uma tag de fim, com o conteúdo inserido no meio:

              <tag>Conteúdo...</tag>

O elemento HTML vai da tag de abertura até a tag de fechamento:

              <p>Meu primeiro parágrafo.</p>
Tag de abertura Conteúdo Tag de fechamento
<h1> Meu primeiro cabeçalho </h1>
<p> Meu primeiro parágrafo. </p>
<br>    

Elementos sem conteúdo são chamados elementos vazios. Elementos vazios não possuem tag de fechamento, assim como o elemento <br> (que indica a quebra de linha).

Elementos Aninhados

Elementos HTML podem ser aninhados (elementos podem conter elementos).

Todo documento HTML consiste de elementos aninhados.

Este exemplo contém quatro elementos:

Exemplo

              <!DOCTYPE html>
              <html>
                <body>

                  <h1>Meu primeiro cabeçalho</h1>
                  <p>Meu primeiro parágrafo.</p>

                </body>
              </html>

Exemplo Explicado

O elemento <html> define o documento inteiro.

Contém uma tag de abertura <html> e uma tag de fechamento </html>

O conteúdo do elemento é outro elemento (o elemento <body>).

              <html>
                <body>

                  <h1>Meu primeiro cabeçalho</h1>
                  <p>Meu primeiro parágrafo.</p>

                </body>
              </html>

O elemento <body> define o corpo do documento.

Contém uma tag de abertura <body> e uma tag de fechamento </body>

Seu conteúdo são dois outros elementos (<h1> e <p>).

              <body>

                <h1>Meu primeiro cabeçalho</h1>
                <p>Meu primeiro parágrafo.</p>

              </body>

O elemento <h1> define um cabeçalho.

Contém uma tag de abertura <h1> e uma tag de fechamento </h1>.

O conteúdo do elemento é: Meu primeiro cabeçalho.

              <h1>Meu primeiro cabeçalho</h1>

O elemento <p> define um parágrafo.

Contém uma tag de abertura <p> e uma tag de fechamento </p>.

O contéudo do elemento é: Meu primeiro parágrafo.

              <p>Meu primeiro parágrafo.</p>

Elementos Vazios

Elementos sem contéudo são chamados Elementos Vazios.

<br> é um elemento vazio sem tag de fechamento (<br> define uma quebra de linha).
