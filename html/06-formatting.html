Elementos de Formatação

HTML define elementos especiais para destacar textos com um significado especial.

HTML utiliza elementos como <b> e <i> para formatar a saída: texto negrito ou itálico.

Elementos de formatação foram feitos para exibir tipos especias de textos:

  • <b> - Texto negrito
  • <strong> - Texto importante
  • <i> - Texto itálico
  • <em> - Texto enfatizado
  • <mark> - Texto marcado
  • <small> - Texto pequeno
  • <del> - Texto tachado
  • <ins> - Texto sublinhado
  • <sub> - Texto subscrito
  • <sup> - Texto sobrescrito

Texto Negrito

O elemento <b> define um texto negrito, sem mais importância.

O elemento <strong> define um texto negrito com importância semântica.

Texto Itálico

O elemento <i> define um texto itálico, sem mais importância.

O elemento <em> define um texto itálico com importância semântica.

Nota: Navegadores exibem <strong> como <b>, e <em> como <i>. Entretanto, há uma diferença no significado dessas tags: <b> e <i> definem texto negrito e itálico, mas <strong> e <em> significam que o texto é importante.
