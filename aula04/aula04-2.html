Estilização

O atributo style é utilizado para alterar o estilo do texto.

style tem a seguinte sintaxe:

Exemplo

              <tag style="propriedade:valor;">

Cor de Fundo

A propriedade background-color define a cor de fundo do elemento.

Exemplo

              <!DOCTYPE html>
              <html>
                <head>
                  <title>Cor de Fundo</title>
                  <meta charset="utf-8">
                </head>
              
                <body style="background-color:powderblue;">
                  <h1>Isto é um cabeçalho</h1>
                  <p>Isto é um parágrafo</p>
                </body>
              </html>

Cor do Texto

A propriedade color define a cor do texto no elemento.

Exemplo

              <h1 style="color:blue;">Isto é um cabeçalho</h1>
              <p style="color:red;">Isto é um parágrafo</p>

Fonte

A propriedade font-family define a fonte a ser usada no elemento.

Exemplo

              <h1 style="font-family:verdana;">Isto é um cabeçalho</h1>
              <p style="font-family:courier;">Isto é um parágrafo</p>

Tamanho da Fonte

A propriedade font-size define o tamanho da fonte no elemento.

Exemplo

              <h1 style="font-size:32px;">Isto é um cabeçalho</h1>
              <p style="font-size:1em;">Isto é um parágrafo</p>

Alinhamento do Texto

A propriedade text-align define o alinhamento horizontal do texto.

  • left alinha o texto à esquerda
  • center centraliza o texto
  • right alinha o texto à direita
  • justify distribui o texto uniformemente entre as margens

Exemplo

              <!DOCTYPE html>
              <html>
                <head>
                  <title>Alinhamento de Parágrafos</title>
                  <meta charset="utf-8">
                </head>
              
              
                <body>
                  <p style="text-align:left;">Este parágrafo está à esquerda</p>
                  <p style="text-align:center;">Este parágrafo está centralizado</p>
                  <p style="text-align:right;">Este parágrafo está à direita</p>
                  <p style="text-align:justify;">Este parágrafo está justificado</p>

                </body>
              </html>

Texto Pré-Formatado

A tag <pre> define um texto pré-formatado.

O texto no elemento <pre> é exibido em fonte monoespaçada (geralmente Courier), e mantém tanto os espaços como quebra de linhas.

Exemplo

              <!DOCTYPE html>
              <html>
                <head>
                  <title>Texto Pré-Formatado</title>
                  <meta charset="utf-8">
                </head>
              
              
                <body>
                  <h1>Texto sem pré-formatação:</h1>
                  <p>
                    Minha terra tem palmeiras,
                    Onde canta o sabiá;
                    As aves que aqui gorjeiam,
                    Não gorjeiam como lá.
                  </p>

                </body>
              </html>

Exemplo

              <!DOCTYPE html>
              <html>
                <head>
                  <title>Texto Pré-Formatado</title>
                  <meta charset="utf-8">
                </head>
              
              
                <body>
                  <h1>Texto pré-formatado:</h1>
                  <pre>
                    Minha terra tem palmeiras,
                    Onde canta o sabiá;
                    As aves que aqui gorjeiam,
                    Não gorjeiam como lá.
                  </pre>

                </body>

Resumo

  • style dá estilo aos elementos
  • background-color define a cor de fundo do elemento
  • color define a cor do texto
  • font-family define a fonte do texto
  • font-size define o tamanho da fonte do texto
  • text-align define o alinhamento do texto
  • A tag <pre> insere um texto pré-formatado
